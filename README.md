# HPCDropOne

## Introduction

Find enclosed in this repository a set of a ansible playbooks with functionailtiy to satisfy requirements 1.1.1, 1.1.2, 1.1.4, 1.3 on Azure and AWS. 

## Deliverable 1.1.1

In this deliverable a virtual application from the hyperscaler's catalogue is deployed as a virtual machine. The virtual application being deployed in a Cisco CSR 1000V. This device is configured with simple Cisco IOS configuration to enable natting from the HPC private network out to the internet.
